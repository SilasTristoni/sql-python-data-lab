# Exemplo de execução

## Objetivo

Este exemplo mostra como executar o projeto **SQL Python Data Lab** localmente e conferir o resumo inicial gerado a partir dos dados do arquivo `data/sample_sales.csv`.

## Comando

Na raiz do projeto, execute:

```bash
python -m src.main
```

## Saída esperada

Um exemplo de saída no terminal:

```text
=== SQL Python Data Lab ===
Total de pedidos: 10
Faturamento total: R$ 13014.50
Ticket médio: R$ 1301.45
Produto mais vendido: Mouse

Faturamento por categoria:
- Eletronicos: R$ 10397.00
- Moveis: R$ 1898.00
- Perifericos: R$ 719.50
```

> Observação: os valores podem mudar conforme o arquivo CSV for alterado.
